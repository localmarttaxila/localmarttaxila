About Local Mart Taxila

Local Mart is a digital marketplace designed specifically for the residents and students of the Taxila and Wah Cantt area. Our mission is to bridge the gap between traditional street vendors and modern consumers.

Project Goals

  • Support local shop owners in Main Bazaar, UET Area, and GT Road.
  • Provide a simple, serverless ordering system for students.
  • Promote the cultural heritage items of Khanpur Road.

Our Technology

This is a functional static web application built using HTML5, CSS3, and JavaScript. It uses Formspree for order management and GitHub Pages for hosting.

Developed as an ICT Class Project - 2025
